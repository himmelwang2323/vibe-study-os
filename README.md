# Vibe Study OS

一个轻量的学习管理系统，用来把长期目标、学期任务、日/周复盘和悬浮学习计时放到同一个工作台里。

## 功能

- 长线目标推进：记录目标、期限、推进理由，并拆成可勾选的小目标。
- 学期任务管理：按课程/主题、周次、截止日期和工作量管理任务。
- 日周学习复盘：记录学习内容、专注时长、能量状态和调整思路。
- 悬浮学习计时器：选择当前工作内容，开始/暂停计时，结束后自动进入日复盘。
- 本地自动保存：数据保存在浏览器 localStorage 中，适合快速试用和继续迭代。

## 使用

直接打开 `index.html` 即可使用。发布到 GitHub Pages 后，也可以通过 Pages 链接访问。

## 桌面版

桌面版使用 Electron 包一层本地窗口，保留网页中的所有数据和计时功能。

```bash
electron_config_cache=.electron-cache npm install
npm run desktop
```

当前桌面版会在关闭窗口时隐藏到托盘，再次点击托盘可打开。正式打包安装包可以后续接入 `electron-builder`。
